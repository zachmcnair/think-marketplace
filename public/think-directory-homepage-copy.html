Option A — Casual

Think Directory is a showcase of the best agents, tools, and apps being built on the Think Protocol — plus the builders behind them.

We're not a marketplace (yet). Think of it as a window into what's possible when AI is something you own, not rent.

Want to be featured? You'll need to contribute something real. That's the only rule.

Option B — Polished

Think Directory is the official showcase for projects and builders in the Think ecosystem.

Here you'll find autonomous agents, developer tools, interfaces, and applications — all built on the open protocol for user-owned AI.

This is a contributor-only showcase. To be featured, builders must actively participate in the Think community.
